## Overview

Inventory Manager is a full-stack warehouse inventory management system designed to help organizations efficiently monitor inventory, manage shipments, coordinate employees, and track business operations through a centralized database.

The application supports multiple user roles, allowing managers, employees, suppliers, and accountants to interact with the system based on their responsibilities. By combining database management, real-time communication, and role-based access control, the project simulates a real-world inventory management platform.

This project provided extensive experience in collaborative software engineering, full-stack web development, database design, Agile development practices, and team-based application development.

---

## Project Objective

The objective of this project was to design and develop a scalable inventory management platform capable of:

- Managing warehouse inventory through full CRUD operations
- Tracking incoming and outgoing shipments
- Supporting multiple user roles with different permissions
- Providing real-time notifications and communication
- Maintaining inventory information within a centralized SQL database
- Applying modern software engineering practices throughout the development lifecycle

---

## Project Features

- Inventory management (Create, Read, Update, Delete)
- Role-based user authentication
- Shipment scheduling and tracking
- Employee shift management
- Supplier shipment management
- Real-time notifications
- Group chat using WebSockets
- CSV import/export functionality
- SQL database integration
